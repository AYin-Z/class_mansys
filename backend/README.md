# 区队管理系统后端服务

## 技术栈
- Node.js + Express
- MySQL
- JWT 认证
- 微信云托管

## 项目结构
```
backend/
├── app.js              # 主应用
├── config/
│   └── database.js     # 数据库配置
├── controllers/        # 控制器
├── middleware/         # 中间件
├── models/             # 数据模型
├── routes/             # 路由
├── utils/              # 工具函数
├── package.json
├── .env
└── Dockerfile          # 微信云托管部署文件
```

## 核心功能
1. 用户认证（登录、注册）
2. 请假管理
3. 通知管理
4. 班费管理
5. 作业管理
6. 公告资源管理
7. 相册管理
8. 心理干预
9. 擂台挑战
10. 投票系统
11. 建议箱
12. 抽奖活动
13. 积分系统

## 部署步骤
1. 准备微信云托管环境
2. 配置数据库
3. 部署后端服务
4. 前端配置API地址

## 注意事项
- 所有API接口需要JWT认证
- 敏感操作需要权限检查
- 数据传输需要加密
